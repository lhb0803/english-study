# English Study

영어 뉴스와 블로그로 공부하는 가벼운 웹앱.

**🔗 https://english-study-one.vercel.app**

매주 **월요일 오전**에 CNN, BBC, NYT, The Guardian, The Verge, Seth Godin, Wait But Why 등 16개 소스에서 최신 글을 모아 **5개**를 추천합니다. 각 아티클은 Claude가 학습자 친화적인 영어로 다시 써 주고, 핵심 단어 5개도 함께 제공돼요.

## 기능

### 📰 목록 페이지
- 주제(경제·시사·기술·엔터·음악·영화·문화·라이프·스포츠·과학)가 서로 다른 5개 아티클
- 한국어 요약 · 난이도(CEFR A2~B2) · 예상 독해 시간 표시
- 카드에서 바로 저장 가능

### 📖 상세 페이지
- Claude가 다시 쓴 **~600단어 학습자용 영어 본문**
- 🔊 **듣기 버튼** — 브라우저 내장 음성으로 본문 재생 (재생/일시정지/이어듣기)
- **Key Vocabulary** — 핵심 단어 5개 (뜻 + 예문)
- **한 줄 요약** — 본문 아래 한국어로 핵심을 다시 확인
- 상단에 스크롤 진행 바
- **스크롤로 끝까지 읽으면 자동 완독 처리 🎉**
- 원문 링크로 이동 가능

### 👤 마이 페이지
- 🔥 **연속 학습일** (스트릭)
- 📚 **완독 수 / 뱃지 수**
- **마일스톤 뱃지** 9종
  - 첫 완독, 5/10/25/50 완독
  - 3일/7일/30일 연속 학습
  - 다방면 탐험가 (서로 다른 5개 주제 완독)
- 저장한 아티클 / 완독한 아티클 목록

### 🔒 프라이버시
- **로그인 없음, 계정 없음**
- 모든 학습 기록은 브라우저 localStorage에 저장 — 서버로 아무것도 보내지 않아요
- 다른 기기에서는 기록이 공유되지 않는다는 점만 참고!

## 왜 만들었나요

기존 영어 공부 앱은 대부분:
- 너무 쉬운 예문 위주이거나
- 너무 어려운 원문 기사를 그대로 던져주거나
- 광고/결제가 많거나

그래서 **학습자 난이도에 맞춘 실제 뉴스**를 주 1회 골라 주는 가벼운 도구를 만들었어요. 매일이 아니라 주 1회인 건, 꾸준히 다 읽을 수 있는 양만 주기 위해서예요.

## 사용 팁

1. **주 1회 추천을 놓치지 마세요.** 월요일 아침에 새 5개가 올라와요.
2. **저장 기능**을 적극 활용하세요. 시간이 없을 땐 일단 저장해두고 나중에 읽어도 스트릭이 유지돼요.
3. **뱃지를 노리세요.** 서로 다른 주제를 번갈아 읽으면 "다방면 탐험가" 뱃지를 받을 수 있어요.
4. **연속 학습일이 끊기는 게 싫다면**, 하루에 짧은 글 하나라도 끝까지 스크롤해서 완독 처리하세요.
